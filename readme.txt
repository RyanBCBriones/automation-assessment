// Automation testing prerequisites:
1. Install python ie. version 2.7
2. Install python package manager (pip)
3. Import Selenium libraries
4. Install robotframework and git
5. Download and install browser driver compatible to the browser version ie. Chromedriver for Chrome, Geckodriver for Firefox.
6. Download text editor ie. VScode, sublime-text

// Test Setups

7. Create a folder as the main repository for the robot test files ex. robot.
8. Create a separate folder inside the main repo for the keywords.
9. Create a separate folder inside the main repo for the variables/constants.
10. You may create a a separate folder for the test suite or just save it on the main repo.
11. Create and save your files with .robot file extension on their respective folders

// Running the test
12. Go to the main directory/repo/master branch ex. Login
13. Use "robot" command or "pybot" to run the test ex. robot login.robot or pybot login.robot
